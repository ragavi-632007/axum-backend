# Axum Backend

This project is a backend server built using [Axum](https://github.com/tokio-rs/axum), a web application framework for Rust. It is designed to provide a robust, scalable, and efficient API service.

## Features

- Built with Rust for safety and performance
- Uses Axum for routing and middleware
- Modular code structure for maintainability
- Example authentication and ticket routes

## Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable recommended)

### Running the Server

```sh
cargo run
```

### Running Examples

```sh
cargo run --example quick_dev
```

## Project Structure

- `src/` - Main source code
- `src/web/` - Web routes and middleware
- `examples/` - Example usage
